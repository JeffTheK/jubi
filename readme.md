# jubi

Simple terminal calendar

## Installation

Clone the repo:

```sh
git clone https://github.com/JeffTheK/jubi
```

Then run `make install` in the project directory

```sh
make install
```

## Usage

Just run jubi in terminal

## Building

```sh
make install
```

### Tests

```sh
make test
```
